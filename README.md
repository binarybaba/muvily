# Muvily for Chrome
Chrome extension to get IMDB rating as push notification
Available on the [webstore][webstore]
### Version
0.0.0.1

## But how?!!
  - Highlight movie name on webpage
  - Right click
  - Select 'Get Movie details'

## Dependencies
  - [jQuery] for le stuff
